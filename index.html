Leaflet directive for AngularJS

Simple to use AngularJS directive to work with leaflet library and Open Street Maps inside your applications.

Code on Github Download

This is a very simple AngularJS directive developed to embed and interact with an map managed by the leaflet map library. Its not intended as a complete or full-of-features library, although its a simple and easy to use directive, which has been fun to develop and has a good starting point to learn to develop a directive to the great AngularJS library.

The code of the library and this page has been inspired on the Angular Google Maps library by nlaplante, and most of the features are based on the same features developed on that library.

Usage

Before using angular-leaflet-directive, you must include some javascript files to your main HTML: the main AngularJS library, the Leaflet library and the angular-leaflet-directive.js script. Something like this:

<script src="/path/to/leaflet.js"></script>
<script src="/path/to/angular.min.js"></script>
<script src="/path/to/angular-leaflet-directive.js"></script>
<script src="/path/to/your-angular-controller.js"></script>

Also, you must include the CSS files of leaflet

<link rel="stylesheet" href="/path/to//leaflet.css" />

And that's all. Now you can use the directive with the custom tag <leaflet> and some attributes, let's see them on the examples.

Simple Demo

You can see the map embeded on the right-side of this page. This is accomplished width this directive:

<leaflet></leaflet>

You can also see a simple code of this feature in the simple example demo.

More complex Demo

You can interact with a leaflet marker with this code on your HTML:

<leaflet marker="marker" center="center" zoom="zoom" draggable="true"></leaflet>

And this code on your controller:

angular.extend($scope, {
    center: { lat: 40.094882122321145, lng: -3.8232421874999996 },
    marker: { lat: 40.094882122321145, lng: -3.8232421874999996 },
    message: "Drag me to your node position",
    zoom: 4
});

See how the form inputs change with the map interaction and viceversa.
